## Data

### Storing states:

#### Numbering:
```
#####
#678#
#345#
#012#
#####
```

#### Local boards:
 - `0` = Blank
 - `1` = X
 - `2` = O

#### Global boards:
 - `0` = Local
 - `1` = Big X
 - `2` = Big O
  
### Structure:
Two dimensional array, first dimension global, second dimension local board.\
`game[9][9];`
